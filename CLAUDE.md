# CLAUDE.md

이 파일은 이 디렉토리에서 작업할 때 지켜야 할 규칙을 정의합니다. 자세한 요구사항은 `PRD.md`를 참고하세요.

## 규칙

1. **정적 사이트만 사용** — API 호출, 서버, 데이터베이스 등 백엔드 연동을 금지합니다. 모든 동작은 클라이언트 사이드에서만 처리합니다.
2. **단일 파일 구성** — 결과물은 `index.html` 한 파일로 완결되어야 하며, CSS와 JS는 해당 파일 안에 인라인(`<style>`, `<script>`)으로 포함합니다. 별도의 `.css`/`.js` 파일을 만들지 않습니다.
3. **모든 텍스트는 한국어**로 작성합니다.
4. **반응형 필수** — 모바일과 데스크톱 등 다양한 화면 크기에서 정상적으로 보여야 합니다.
