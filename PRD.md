# PRD: 프리랜서/컨설턴트 포트폴리오 랜딩페이지

## 1. 개요
- **형태**: 정적 사이트, `index.html` 단일 파일 (HTML/CSS/JS 인라인 포함, 외부 빌드 도구 없음)
- **목적**: 프리랜서/컨설턴트 본인을 소개하고, 방문자가 SNS(인스타/유튜브 등) 프로필로 이동하도록 유도
- **타깃 사용자**: 협업/작업을 검토 중인 잠재 고객 또는 파트너
- **톤앤매너**: 밝고 친근한/캐주얼 (화이트 계열 배경, 여백 넉넉, 부드러운 색감과 둥근 모서리 등 친근한 인상)
- **언어**: 한국어

## 2. 핵심 CTA
- 방문자가 최종적으로 취해야 할 행동: **SNS 프로필로 이동** (예: 인스타그램, 유튜브 등)
- 실제 URL은 현재 미정 → 플레이스홀더 링크(`#`)로 두고 `TODO` 주석으로 표시, 추후 교체
- CTA 버튼은 히어로 섹션과 하단(마무리) 섹션 등 최소 2곳에 배치 권장

## 3. 페이지 구성 (섹션)

### 3.1 히어로 (Hero)
- 이름/직함 (예: "OOO · 프리랜서 컨설턴트") — 실제 이름 미정, 플레이스홀더 텍스트 사용
- 한 줄 소개 문구 (캐치프레이즈)
- 짧은 설명 (본인이 제공하는 가치)
- 주요 CTA 버튼 (SNS 프로필 이동)
- 프로필 이미지/일러스트 자리 (실제 이미지 없을 시 플레이스홀더 처리)

### 3.2 보유 스킬/역량 (Skills)
- "주요 기능 소개" 섹션을 스킬/역량 리스트로 구성
- 아이콘 + 스킬명 + 짧은 설명 형태의 카드 나열 (예: 3~6개 항목)
- 실제 스킬 목록 미정 → 플레이스홀더 항목 3~6개로 예시 구성

### 3.3 FAQ (작업/협업 문의 관련)
- 아코디언(접고 펼치기) 형태
- 예상 질문 예시 (플레이스홀더로 작성, 추후 실제 내용으로 교체):
  - 협업/작업 문의는 어떻게 하나요?
  - 응답 소요 시간은 얼마나 되나요?
  - 작업 단가/견적은 어떻게 산정되나요?
  - 원격으로 진행이 가능한가요?
- 최소 4개 내외 항목

### 3.4 푸터 / 마무리
- 보조 CTA 버튼 (SNS 프로필 이동 재노출)
- 저작권 문구 등 간단한 마무리 정보

## 4. 디자인 가이드
- **색상**: 밝은 화이트/라이트 배경 중심, 포인트 컬러 1~2개 (파스텔 또는 생동감 있는 캐주얼 톤) — 구체적 컬러 팔레트는 미정이므로 구현 시 임의로 친근한 톤 선정
- **타이포그래피**: 가독성 좋은 산세리프, 부드럽고 친근한 인상의 폰트
- **레이아웃**: 여백 넉넉, 둥근 모서리(버튼/카드), 부드러운 그림자 등 캐주얼한 UI 요소
- **반응형**: 모바일/데스크톱 모두 대응 (단일 컬럼 ↔ 멀티 컬럼 전환)

## 5. 기술 요구사항
- 순수 HTML/CSS/JS, 파일 하나(`index.html`)로 완결
- 외부 라이브러리/프레임워크 의존 최소화 (필요 시 CDN 방식 허용 여부는 별도 확인)
- 이미지 등 별도 에셋 없이 우선 텍스트/CSS 중심 또는 SVG 인라인 활용
- 접근성(시맨틱 태그, 대비 등) 기본 준수

## 6. 미확정 사항 (추후 확인 필요)
- 실제 이름/직함/프로필 문구
- 실제 SNS 프로필 URL
- 실제 보유 스킬 목록
- 실제 FAQ 질문/답변 내용
- 프로필 사진 등 이미지 에셋 유무
- 구체적인 브랜드 컬러 지정 여부
- 외부 CDN(폰트, 아이콘 등) 사용 허용 여부

## 7. Out of Scope
- 별도 문의 폼 백엔드 연동 (CTA는 외부 링크 이동 방식이므로 서버 로직 불필요)
- 다국어 지원
- 요금제/가격 섹션, 후기/신뢰 요소 섹션 (이번 범위에서는 제외)
